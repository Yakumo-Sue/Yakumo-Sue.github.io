00. C++ 学习阶段

C++ 学习路线​

第一阶段：入门​

  • 计算机体系架构基础：
    • 图灵机、冯诺依曼体系架构
    • 计算机总线
    • 计算机信息表示与存储
    • CPU与指令集
    • 程序基础概念
    • 编程语言类型
  • Linux 基础：
    • 常用命令
    • 权限管理
  • C 语言基础：
    • 数据类型
    • 宏
    • 数据运算
    • 数组
    • 输入输出
    • 分支与循环
    • 函数
    • 指针
    • 结构体/联合体/枚举/sizeof
    • 文件操作
  • 计算机网络基础：
    • 局域网技术
    • 集线器/交换机/路由器
    • OSI七层模型 / TCP/IP四层模型
    • IP协议
    • TCP/UDP
    • DNS
    • HTTP

第二阶段：基础​

  • C++ 基础：
    • 面向对象基础
    • 友元
    • static/const
    • 虚函数
    • 指针与引用
    • RAII
    • STL
  • 数据结构与算法：
    • 栈
    • 队列
    • 树/二叉树/二叉搜索树
    • 链表/双向链表/交叉链表
    • 哈希表
    • 十大排序算法
    • 二分查找
  • 编译原理：
    • 预处理
    • 编译/链接
    • 可执行文件
    • makefile
    • CMake
    • gcc / g++

第三阶段：进阶​

  • 操作系统基础：
    • 进程与线程
    • 进程地址空间布局
    • 虚拟内存
    • 内核态与用户态
    • 系统调用
    • 中断与异常
  • C++ 进阶：
    • 异常处理
    • 泛型编程
    • 内存对象模型
    • RTTI
    • 多态实现原理
    • 四大类型转换
  • C++ 11：
    • 右值引用
    • 自动类型推导
    • unordered_map
    • thread
    • lambda
    • for each
    • 智能指针
  • 线路堆栈：
    • 内存对齐
    • 线程栈布局
    • 函数调用过程
    • 栈溢出攻击
    • 堆内存分配算法
    • new与delete原理
    • delete[] 原理
  • 软件调试：
    • GDB
    • 反汇编
    • 软件调试原理

第四阶段：扩展​

  • 网络编程：
    • socket
    • IO模型（select/poll/epoll)
  • 多线程多进程编程：
    • fork机制
    • pthread
    • 线程同步
    • 线程局部存储TLS
  • 进程间通信：
    • Signal
    • 信号量
    • 命名管道/匿名管道
    • 共享内存
    • 条件变量
  • 数据结构与算法进阶：
    • 动态规划
    • 布隆过滤器
    • 贪心算法
    • KMP
    • 二叉树相关
    • TOP K问题
  • 设计模式：
    • 单例模式
    • 工厂模式
    • 代理模式
    • 观察者模式
    • 适配器模式
    • 桥接模式
    • 责任链模式

第五阶段：工程化​

  • 计算机底层技术：
    • CPU：
      • 缓存
      • 中断与异常
      • 线程亲和性
      • NUMA架构
      • 特权级
    • I/O
      • 零拷贝技术
      • DMA
      • 内存映射
  • RPC与序列化技术：
    • gRPC与Protobuf
    • Thrift
    • RapidJSON
    • AVRO
  • 工程组件：
    • MySQL
    • Redis
    • nginx
    • docker
    • kafka
  • 软件安全
    • 缓冲区溢出
    • 释放后使用
    • 空指针攻击
    • 悬空指针与野指针
    • 哈希算法
    • 加密与解密

4.清除默认样式&选择器&显示&背景

清除浏览器默认样式​

浏览器的默认样式是固定的，我们作为开发者每次手动清除浏览器默认样式。

但是，其实很多开发者已经进行了总结，清除浏览器默认样式的所有代码,拿到代码。每次只要引入该代码，就可以清除浏览器默认样式了，不需要每次自己写。

normalize.css： https://github.com/necolas/normalize.css/blob/master/normalize.css

为什么不用通配符来清除浏览器默认样式?​

因为通配符的作用是找到页面中的所有标签并清除浏览器默认样式，这样效率很低，导致页面性能低，所以我们宁可一个个的去清除有必要清除的绝不做宁可错杀不可放过的事情。

复合选择器​

  • 后代选择器 标志是空格 作用是: 选择到所有的后代元素 并给后代元素添加样式

    <!DOCTYPE html>
    <html lang="en">

    <head>
        <meta charset="UTF-8">
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
        <meta http-equiv="X-UA-Compatible" content="ie=edge">
        <title>Document</title>
    </head>
    <style>
        .div1 span {
            width: 200px;
            height: 200px;
        }

        .div2 span {
            width: 200px;
            height: 200px;
        }
    </style>

    <body>
        <div class="div1">
            <span>1</span>
            <span>1</span>
            <span>1</span>
            <span>1</span>
        </div>

        <div class="div2">
            <span>1</span>
            <span>1</span>
            <span>1</span>
            <span>1</span>
        </div>
    </body>

    </html>

  • 子代选择器 标志是 > 作用是: 选择到所有的亲儿子元素 并给指定的亲儿子添加样式

    <!DOCTYPE html>
    <html lang="en">
    <head>
        <meta charset="UTF-8">
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
        <meta http-equiv="X-UA-Compatible" content="ie=edge">
        <title>Document</title>
    </head>
    <style>
        .box > p {
            color : red;
        }
    </style>
    <body>
        <div class="box">
            <p>
                1
            </p>

            <div>
                <p>2</p>
            </div>
        </div>
    </body>
    </html>

  • 并集选择器 标志是 , 作用是: 给一组拥有相同样式的元素添加样式

    <!DOCTYPE html>
    <html lang="en">
    <head>
        <meta charset="UTF-8">
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
        <meta http-equiv="X-UA-Compatible" content="ie=edge">
        <title>Document</title>
    </head>

    <style>
        span,
        p,
        h1,
        div b{
            color:red;
        }
    </style>
    <body>

        <span>a</span>
        <p>b</p>
        <h1>c</h1>
        <div>
            <b>d</b>
        </div>

    </body>
    </html>

  • 伪类选择器 标志是 : 作用是: 特殊选择元素(:first-child :last-child) 特殊效果(:hover)

    <!DOCTYPE html>
    <html lang="en">

    <head>
        <meta charset="UTF-8">
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
        <meta http-equiv="X-UA-Compatible" content="ie=edge">
        <title>Document</title>
    </head>
    <style>
        a {
            color: red;
        }

        a:link{
            color:rosybrown
        }

        a:hover{
            color:green,
        }

        a:active{
            color:sandybrown
        }
    </style>
        
    <body>
        <a href="#"百度>百度</a>
    </body>

    </html>

元素的显示模式​

块级元素​

  • 举例: div、h1-h6，p，ul，ol，li，table

  • 特点: 独占一行 可以设置宽高外边距内边距 默认宽度是父元素的100%

  • 注意: 文字类的块元素不能套其他块元素

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <meta http-equiv="X-UA-Compatible" content="ie=edge">
    <title>Document</title>
</head>
<style>
    div{
        width: 500px;
        height: 200px;
        border: 1px solid chartreuse;
        padding: 2px;
        margin:2px;
    }
</style>
<body>
    
    <!-- 块元素 div h1~h6 p ul ol li -->
    <!-- 块元素的特点: 1.独占一行  2.可以设置任何属性(宽 高 内边距 外边距) 3.默认宽度是父元素宽度的100% -->
    <div>
        ssaf
    </div>

    <div>
        fadafa
    </div>

</body>
</html>

行内元素​

  • 举例: span，a，strong，i，b，small

  • 特点: 一行显示多个，没有宽高没有上下 padding 和 margin 默认宽度就是内容的宽。行内元素只能嵌套文本或其他行内元素

  • 注意: 链接内不能再放链接 a 链接特殊情况，不过为了不出问题 建议大家把a标签转换为块元素

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <meta http-equiv="X-UA-Compatible" content="ie=edge">
    <title>Document</title>
</head>
<style>
    span{
        /* 行内元素没有上下的padding和margin */
        padding-top: 20px;
        margin-top: 20px;
    }
</style>
<body>
    <!-- 行内元素: span a strong b s i em del  -->
    <span>张三</span>
    <a href="">里斯</a>
    <strong>王五</strong>
    <b>赵六</b>
    <s>天启</s>
    <i>阿凡达</i>
    <em>发顺丰</em> 
</body>
</html>

行内块元素​

  • 举例: img,input,td

  • 特点: 一行显示多个 有宽高上下 padding 和 margin 默认宽度就是内容的宽度

  • 注意: 不需要管嵌套 因为 img 是单标签 inpu t是单标签 没有嵌套这一说

  • 如何去除行内块之间的距离？

    给行内块元素的父元素添加一句 css 代码: font-size: 0;

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <meta http-equiv="X-UA-Compatible" content="ie=edge">
    <title>Document</title>
</head>
<style>
    body {
        background-color: cornflowerblue;
        font-size: 0;
    }

    img{
        width: 150px;
        /* auto是随着宽度的变化 ,  ....高度也自动变化 */
        height:auto;
    }
</style>
<body>
    <!-- 行内块元素: 一显示多个  元素之间有空隙  默认宽度是内容的速度 可以设置宽高-->
    <img src="" alt="">
    <img src="" alt="">
    <img src="" alt="">
    <img src="" alt="">
</body>
</html>

显示模式的转换​

  • 其他转为块： display: block
  • 其他转为行内： display: inline
  • 其他转化为行内块： display: inline-block

注意: 所有人在进行模式转换的时候 请把display作为第一个属性

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <meta http-equiv="X-UA-Compatible" content="ie=edge">
    <title>Document</title>
</head>
<style>
    p{
        display:inline;
    }

    span{
        display:block;
        width:200px;
        height:200px;
        background-color: crimson;
    }

    input{
        display: block;
    }
    
</style>
<body>
        <p>你</p>
        <p>你</p>
        <p>你</p>
        <p>你</p>

    <span>我</span>

    <input type="text">
    <input type="text">
    <input type="text">
    <input type="text">
</body>
</html>

CSS背景属性​

背景颜色 background-color​

  • transparent 完全透明
  • rgba(红色,绿色,蓝色,透明度) 0-1 0为完全透明 1为完全不透明

背景平铺：

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <meta http-equiv="X-UA-Compatible" content="ie=edge">
    <title>Document</title>
</head>
<style>
    html,
    body{
        margin:0;
        padding:0;
        /* 意思是 和浏览器的宽高一样大 */
        width:100%;
        height:100%;
    }

    body{
        background-image: url();
        background-repeat: no-repeat;
        background-size: 100%;
        background-attachment: scroll;
    }

    p{
        font-size: 30px;
    }

</style>
<body>
    <p></p>
</body>
</html>

背景尺寸：

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <meta http-equiv="X-UA-Compatible" content="ie=edge">
    <title>Document</title>
</head>
<style>
    /* background-size 是用来设置背景图片大小的
    background-size:宽度  高度; 具体的数字
    background-size: cover contain */

    .box{
        width:200px;
        height:200px;
        background-image: url();
        background-repeat: no-repeat;
        border:1px solid #ccc;

        /* cover的作用是填满盒子  无论图片是否完全显示 */
        background-size: cover;

        /* cover的作用是把图片显示全  不管是否填满盒子 */
        background-size: contain;
    }
</style>
<body>
    <div class="box"></div>
</body>
</html>

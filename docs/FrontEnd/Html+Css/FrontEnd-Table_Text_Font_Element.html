表格​

  1. 表格基本组成

表格基本组成​

  • table 标签 用于告诉浏览器 我接下来要创建表格了
  • tr 用于创建表格的行
  • td 用于创建表格的列
  • th 标题单元格
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <meta http-equiv="X-UA-Compatible" content="ie=edge">
    <title>Document</title>
</head>
<body>
    <!-- 创建表格
    table 用于创建表格
    tr 用于创建表格的行 table row
    td 用于创建表格的列 table data -->

    <!-- 需求: 一行 4列的表格-->
<table border="1" cellspacing="0" cellpadding="4">
    <caption>著名诗人</caption>
    <tr>
        <td>李白</td>
        <td>杜甫</td>
        <td>白居易</td>
    </tr>
    <tr>
        <td>李商隐</td>
        <td>李清照</td>
        <td>李煌</td>
    </tr>
</table>
</body>
</html>

表格的属性​

  • border 给table标签和td添加边框

  • cellspacing 设置单元格与单元格之间的距离

  • cellpadding 设置内容与单元格边框的距离

  • rowspan 跨行 从自己算1 跨一行+1

    <!DOCTYPE html>
    <html lang="en">

    <head>
        <meta charset="UTF-8">
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
        <meta http-equiv="X-UA-Compatible" content="ie=edge">
        <title>Document</title>
    </head>

    <body>
        <!-- rowspan row 行 span 跨 -->

        <table border="1" cellspacing="0" cellpadding="20px">

            <tr>
                <td rowspan="2">李白</td>
                <td rowspan="3">唐朝</td>
                <td>静夜思</td>
            </tr>

            <tr>
                <td>杜甫</td>
                <td>唐朝</td>

        </table>

文本缩进(text-indent)​

  • px 像素（Pixel）：相对长度单位。像素 px 是相对于显示器屏幕分辨率而言的。

    • IE 无法调整那些使用 px 作为单位的字体大小；
    • 国外的大部分网站能够调整的原因在于其使用了em 或 rem 作为字体单位；
    • Firefox能够调整 px 和 em，rem，但是 96% 以上的中国网民使用 IE 浏览器(或内核)。

  • em 是相对长度单位。相对于当前对象内文本的字体尺寸。如当前对行内文本的字体尺寸未被人为设置，则相对于浏览器的默认字体尺寸。

    • em 的值并不是固定的；
    • em 会继承父级元素的字体大小。

注意：

em 是一个相对长度单位 em 相对的是自身的 font-size 大小 1em= 1* 自己的 font-size

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <meta http-equiv="X-UA-Compatible" content="ie=edge">
    <title>Document</title>
</head>
<style>

    /* 首航缩进 text-indent */
    /* 单位: px em */

    /* em是一个相对长度单位() em相对的是自身的font-size */
    div{
        /* 应为em相对的是自身的font-size 所以1em=20px */
        font-size: 20px;
        /* 20px * 30px */
        width: 20em;
        height: 20em;
        background-color: slategray;
    }
    /* 缩进 px em */

    p{
        /* text-indent: 1px; */
        /* 2*该文字的字体大小 */
        font-size: 2px;
        text-indent: 5em;
    }

</style>
<body>
    <div></div>
    <h4>
        覅还是发好后i佛i奥i覅和覅i佛啊觉得似乎分化和
    </h4>

    <p>
        发i哈佛i阿红哈佛阿红覅回复害怕和发票发哦覅偶爱金佛i就
    </p>
</body>
</html>

文本装饰 text-decoration​

  • none： 无
  • underline： 下划线
  • overline： 上边线
  • line-through： 删除线
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <meta http-equiv="X-UA-Compatible" content="ie=edge">
    <title>Document</title>
</head>
<style>
    /* 文本装饰:
        属性:text-decoration
        属性值:none 取消所有装饰
               underline 添加下划线
               overline 文本上边线
               line-through 删除线 */

    p{
        text-decoration: none;
        text-decoration: underline;
        text-decoration: overline;
        text-decoration: line-through;
    }           

</style>
<body>
    <p>滑稽大法保平安</p>
</body>
</html>

行高 line-height​

  • 200px 代表行高是200px
  • 200% 代表200%*自身的字体大小得出的像素值
  • 2 代表2*自身的字体大小得出的像素值

注意：

行高的作用是让文字垂直居中于盒子 让文字的行高=盒子的高度 

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <meta http-equiv="X-UA-Compatible" content="ie=edge">
    <title>Document</title>
</head>
<style>
    div{
        width:200px;
        height:200px;
        background-color: green;
        text-align: center;
        line-height: 200px;
    }
</style>
<body>
    <div>
        你好
    </div>
</body>
</html>

字体大小 font-size​

就是单位为 px 的一些字体大小

字体种类 font-family​

"Microsoft YaHei" "SimSun"

    <meta name="viewport" content="width=device-width, initial-scale=1.0">

</head>
<style>
    /* 作用:font-size属性用于设置字号 */
    /* 属性: font-size
        值:具体的值  */

    /* 给文字设置字体样式 font-family */

    /* 加粗用的属性叫: font-weight */
    /* normal 默认值(不加粗的)
        bold 定义粗体(加粗的)

伪元素​

伪元素的用途​

  1. 清除浮动
  2. 鼠标移入有一个遮罩

伪元素的用法​

  1. 给真实元素添加
  2. 真实元素 :before 和 真实元素 :after
  3. before 和 after 能不能加上全看 content 有没有写
  4. 给某一个标签添加 before 和 afte 只能给该标签添加一个 before 或一个 after 不能给一个标签添加多个 before 或 after
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <meta http-equiv="X-UA-Compatible" content="ie=edge">
    <title>Document</title>
</head>
<style>
    div{
        width:200px;
        height:200px;
        border:1px solid red;
    }

    div:before{
        /* content的含义: 内容 */
        content:"你好";
        padding:30px;
        border:1px solid red;
    }

    div:after{
        content:"";
        padding:30px;
        background-color: grey;
    }

</style>
<body>
    <!-- 伪:假 -->
    <!-- 伪元素: after before
         伪元素是加在真实元素之上的
    -->
    <div>
        <!-- <span class="before">nihao</span>
        <span class="after"></span> -->
    </div>
</body>
</html>
